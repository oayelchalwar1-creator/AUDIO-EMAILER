Audio Emailer – Team 03

Project Overview

Audio Emailer is an accessible email client designed to assist users who have difficulty reading screens.
The system allows users to listen to emails instead of reading them, enabling hands-free interaction.
The application connects to email providers (Gmail, Yahoo, Outlook), retrieves emails, and reads them aloud using text-to-speech.
